# Day 02

## Visual Editors Research

Started by researching different visual editors and page builders used with React.

### Summary of Options

- Puck
  - Balanced between flexibility and ease of use
  - Native React support
  - Ready-to-use UI with drag-and-drop
  - Good starting point for building a visual editor

- Craft.js
  - Full control over everything
  - Works directly with React components
  - No built-in UI (build everything manually)
  - More complex and developer-heavy

- GrapesJS
  - Powerful but not React-based
  - Works with HTML/CSS
  - Less suitable for modern React apps

- Builder.io / Storyblok
  - Full SaaS solutions
  - Easy to use but limited customization
  - Vendor lock-in

- Tiptap
  - Text editor, not a page builder
  - Not relevant for layout-based editors

## Conclusion

- Craft.js gives full control over the editor and UI
- Suitable for building everything from scratch with full customization

- Puck provides:
  - Faster setup
  - Ready UI
  - Easier starting point

Decision:
- Puck is more suitable for current needs
- Craft.js can be considered later if deeper customization is required
